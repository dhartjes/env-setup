# Configure Brave for Search
Node > 22
Molt

## About
- 1 request per second
- 2000 requests per month

## Steps
1. Create Brave-Search-API account
1. Subscription: Free
1. Usage limit: $5
1. API Key: Create and copy
1. In terminal:

```
moltbot configure --section web

# In interactive:
#   - Set API Key
```
