## Steps
1. Launch Windows Features with Win+r, appwiz.cpl
1. Click Turn Windows Features on or off
1. Enable .NET Framework 4.8 Advanced Services > ASP.NET 4.8
1. Enable Internet Information Services > World Wide Web Services > Application Development Features > .NET Extensibility 4.8
1. Enable Internet Information Services > World Wide Web Services > Application Development Features > ASP.NET 4.8
1. Enable Internet Information Services > World Wide Web Services > Application Development Features > ISAPI Extensions
1. Enable Internet Information Services > World Wide Web Services > Application Development Features > ISAPI Filters
