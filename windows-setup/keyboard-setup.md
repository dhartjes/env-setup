# Keyboard Setup

## Overview
My Anne Pro 2 keyboard requires Obinskit to connect via bluetooth. Settings can be a bit of a pain. Perhaps storing the settings info in this repo will help with keeping the settings synced over time.

## Installation
winget install ObinsKit.ObinsKit

## First run
1. In PowerShell: `cd $env:PROGRAMFILES/ObinsKit/ && .\ObinsKit.exe`
1. Ensure Bluetooth mode switch is in off position
1. Run Firmware Upgrade via ObinsKit UI
1. Connect keyboard via USB
1. Configure for keyboard mode and save config
1. Disconnect USB
1. 
https://manuals.plus/obins/obins-anne-pro-2-manual
