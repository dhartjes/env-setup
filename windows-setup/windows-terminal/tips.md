# Windows Terminal Tips

## Hotkeys
|Hotkey|Command|
|------|-------|
|Ctrl+Shift+P|command pallete|
|Alt+Shift++|new vertical pane|
|Alt+Shift+-|new horizontal pane|
|Ctrl+Shift+t|new tab with default profile|
|Ctrl+Shift+C|copy from the command line|

## Settings
- Export your settings from one device to the next by saving here in this repo. In my current instance, the settings file can be found at C:\Users\Dominic.Hartjes\AppData\Local\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\settings.json
- Open the Settings .json file by holding shift when clicking settings in the main terminal dropdown.
- Set the default startup profile to Ubuntu 
- In Settings > Startup, set "Launch on machine startup" to true
