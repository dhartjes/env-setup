# Default text editor: Microsoft Edit

## Installation
winget install Microsoft.Edit

## Configuration
1. Use windows settings and open App Defaults.
1. Search for .txt and choose new default app using the "Choose an App on your PC" option.
1. Locate Microsoft.Edit. (Currently found in %LocalAppData$/Microsoft/WinGet/Packages/Microsoft.Edit_Microsoft.Winget.Source_8wekyb3d8bbwe/edit-1.2.1-x86_64-windows/edit.exe)
1. Repeat above steps for .json files
